# Running the network engine

1. First run the `Setup.sh` script. It should set the huge pages and ARP entries by pinging, and should not go wrong at all.
2. Then set the corresponding values for each option according to the physical setup, i.e., IP addresses, MAC addresses, etc. Verify that each option is properly configured.
3. Then start the network engine using `Run.sh`.
4. Then start the host and client application as normal.
